# Standar Deviasi

Standard Deviasi merupakan pengukuran untuk mengetahui variasi pada variabel random.
Standard Deviasi menunjukan seberapa jauh nilai data dalam satu set data tersebar dari nilai
nilai rata-rata (mean) dari set data tersebut.
Standard Deviasi sering di aplikasikan dalam bidang seperti keuangan,Machine learning,statiska dsb.

Rumus:

```
σ = √∑(x - μ)^2 / n
```

